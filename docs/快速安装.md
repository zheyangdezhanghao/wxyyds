# 快速安装（小白版）

> 3 分钟搞定，**不会删除聊天记录**。

## 方法一：双击安装（最简单）

1. 下载或 clone 本项目到 Mac
2. **完全退出微信**（菜单栏 → 微信 → 退出微信）
3. 双击 **`一键安装.command`**
4. 若提示「无法打开」：右键 → **打开** → 再点 **打开**
5. 按提示按回车，等待出现 **✅ 安装成功**

## 方法二：一行命令（不用下载 zip）

打开 **终端**（应用程序 → 实用工具 → 终端），粘贴后回车：

```bash
curl -fsSL https://raw.githubusercontent.com/zheyangdezhanghao/wxyyds/main/scripts/bootstrap.sh | bash
```

脚本会自动：
- 下载项目到 `~/wxyyds`
- 检测微信版本
- 安装防撤回 + 多开

## 方法三：手动 clone

```bash
git clone https://github.com/zheyangdezhanghao/wxyyds.git
cd wxyyds
bash install.sh
```

## 安装前 checklist

| 步骤 | 说明 |
|------|------|
| ✅ 退出微信 | 安装时必须完全退出，脚本可帮你自动退出 |
| ✅ 打开一次微信 | 全新 Mac 请先正常打开微信一次再安装 |
| ⚠️ 完整磁盘访问 | 若报权限错误，到 **系统设置 → 隐私与安全性 → 完整磁盘访问权限**，给终端开权限 |

## 安装后能做什么

| 功能 | 怎么用 |
|------|--------|
| 防撤回 | 自动生效，好友撤回消息你仍能看到 |
| 多开 | 终端运行 `open -n /Applications/WeChat.app` |
| 卸载 | 在项目目录运行 `bash uninstall.sh` |

## 常见问题

**Q: 提示微信版本不支持？**  
安装脚本会询问是否自动升级到已适配版本，选 **1** 即可，聊天记录保留。

**Q: 安装后微信打不开？**  
先运行 `bash uninstall.sh` 恢复，检查是否给了终端「完整磁盘访问权限」后重试。

**Q: Intel Mac 和 M 系列芯片？**  
都支持。Intel 默认 patch 模式（最稳）；Apple Silicon 可选更多 Framework 功能。
