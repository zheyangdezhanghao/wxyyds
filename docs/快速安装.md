# 快速安装

> 完整说明见 **[GUIDE.md](GUIDE.md)**。本文仅保留最短路径。

---

## 三步安装（小白）

1. [下载项目](https://github.com/zheyangdezhanghao/wxyyds/archive/refs/heads/main.zip) 或 `git clone https://github.com/zheyangdezhanghao/wxyyds.git`
2. **完全退出微信**
3. 双击 **`一键安装.command`**（被拦截则右键 → 打开）

---

## 一行命令

```bash
curl -fsSL https://github.com/zheyangdezhanghao/wxyyds/raw/main/scripts/bootstrap.sh | bash
```

---

## 推荐命令（按平台）

| 平台 | 命令 |
|------|------|
| Intel 269077 日常 | `bash install.sh --with-framework --yes` |
| 只要防撤回+多开 | `bash install.sh --yes` |
| Apple Silicon | `bash install.sh --yes`（自动引导 32288） |
| 仅检测 | `bash install.sh --check-only` |

---

## 安装后

```bash
bash scripts/smoke-stability.sh
open -n /Applications/WeChat.app   # 多开
```

---

## 版本速查

| 架构 | Build | 微信版本 |
|------|-------|----------|
| Intel | 269077 | 4.1.11.21 |
| Apple Silicon | 32288 | 4.1.5.28 |

聊天记录 **不会被修改**（`~/Library/Containers/com.tencent.xinWeChat`）。
